## 功能测试清单

### ✅ Phase 1: 启动和清单加载

**测试步骤：**
1. 启动程序
2. 观察主窗口是否正常显示
3. 检查底部状态栏显示网络状态（Online/Offline）
4. 确认软件包列表是否加载（应显示 5 个示例软件包）

**预期结果：**
- 主窗口标题：装机 Agent - 软件批量部署工具
- 软件包卡片显示：7-Zip, VS Code, Git, Chrome, Python
- 状态栏显示："已加载 5 个软件包 (在线/离线模式)"

---

### ✅ Phase 2: 网络检测

**测试步骤：**
1. 在线环境下启动程序
2. 观察右下角网络状态显示 "Online"
3. 断开网络连接
4. 重启程序
5. 观察网络状态变为 "Offline"

**预期结果：**
- 在线时：3 秒内完成检测
- 离线时：立即使用本地清单

**测试步骤：**
1. 点击"检查更新"按钮
2. 观察状态栏消息变化

**预期结果（在线）：**
- 状态显示"正在检查更新..."
- 完成后显示"所有软件包已是最新版本"或"发现 X 个更新"

**预期结果（离线）：**
- 状态显示"所有软件包已是最新版本"

### ✅ Phase 4: 软件包选择

**测试步骤：**
1. 勾选/取消勾选软件包卡片左上角的复选框
2. 观察选中状态变化

**预期结果：**
- 复选框状态正确反映
- 默认选中的软件包已勾选

---

### ✅ Phase 5: 安装功能（核心测试）

**测试步骤：**
1. 选中一个或多个软件包
2. 点击"安装选中"按钮
3. 观察状态变化

**预期行为：**
- 加载遮罩显示
- 状态栏显示"正在准备安装 X 个软件包..."
- 如果需要下载，显示下载进度和速度
- 每个软件包卡片显示安装状态
- 完成后状态栏显示"安装完成: 成功 X 个, 失败 X 个"

**注意事项：**
- 示例清单中的软件包哈希值为全 0（占位符），实际下载会校验失败
- 需要用真实的软件包和正确的 SHA256 哈希才能完整测试

---

### ⚠️ Phase 6: 错误处理测试

**测试场景：**

#### 6.1 无网络下载
1. 断开网络
2. 选择没有 LocalPath 的软件包
3. 点击"安装选中"

**预期结果：**
- 状态显示"无可用安装源"或下载失败

#### 6.2 哈希校验失败
1. 下载文件但哈希不匹配

**预期结果：**
- 安装状态显示"校验失败"
- 文件被自动删除

#### 6.3 安装程序不存在
1. 指定不存在的 LocalPath

**预期结果：**
- 状态显示"未找到安装文件"

## 真实软件包测试

### 创建可用的测试清单

修改 `manifest.json`，使用真实的软件包信息：

```json
{
  "id": "7zip-real",
  "name": "7-Zip 24.08",
  "version": "24.08",
  "category": "Utility",
  "publisher": "Igor Pavlov",
  "description": "开源高压缩比文件压缩工具",
  "installer": {
    "type": "exe",
    "url": "https://www.7-zip.org/a/7z2408-x64.exe",
    "hash": "实际的SHA256哈希值（需要计算）",
    "size": 1702400,
    "silentArgs": "/S"
  },
  "detection": {
    "type": "registry",
    "registryKey": "HKEY_LOCAL_MACHINE\\SOFTWARE\\7-Zip",
    "registryValue": "Path"
  }
}
```

### 计算 SHA256 哈希

使用 PowerShell：
```powershell
Get-FileHash -Path "path\to\installer.exe" -Algorithm SHA256 | Select-Object Hash
```

## 已知限制

1. **示例清单的哈希值无效** — 需要替换为真实的 SHA256 值
2. **远程清单 URL 未配置** — `ManifestService.cs` 中的 URL 需要替换
3. **安装需要管理员权限** — 某些软件包（如 Python）需要以管理员身份运行
4. **UI 无进度条** — 当前只有文本状态，下一步需要添加可视化进度条

## 调试技巧

### 查看日志输出
在 Visual Studio 中运行项目，查看"输出"窗口的调试信息。

### 断点调试关键位置
- `MainViewModel.InitializeAsync()` — 启动加载
- `NetworkDetector.IsOnlineAsync()` — 网络检测
- `InstallService.InstallPackageAsync()` — 安装逻辑
- `DownloadService.DownloadPackageAsync()` — 下载逻辑

### 模拟网络环境
- **离线模式**：禁用网络适配器
- **慢速网络**：使用 Windows 网络限速工具
